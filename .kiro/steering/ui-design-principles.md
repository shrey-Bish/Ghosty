# UI Design Principles

## Product Feel
Ghosty should feel like a premium, dark, mobile-first conference companion. It should be fast enough to use in a hallway without turning into another task.

## Principles
- One primary action per screen.
- Recording state must be visibly red and unmistakable.
- Confirmation state must feel calm and reviewable.
- High-value opportunity states use amber.
- Medium-value or completed states use teal.
- At-risk states use red.
- Cards should be dense enough for repeated use, not marketing-heavy.
- All text must fit within compact mobile containers.

## Current Palette
- Background: `#0D1117`
- Surface: `#161B22`
- Amber: `#F59E0B`
- Teal: `#14B8A6`
- Red: `#F43F5E`
